typedef struct xl320_serial_struct XL320_Serial_t

Serial interface structure.

It should be initialized with the corresponding functions according to your target. The XL320 is a digital servo. It communicates in half-duplex (single-wire) UART.

xl320_addHeader2Buff ( uint8_t *  buff)

Add an header to a buffer.

The frame format follows the DYNAMIXEL PROTOCOL 2.0. The header is build on the following pattern :

Header Reserved
0xFF 0xFF 0xFD 0x00
Parameters
buffThe buffer in which the function will add the header. The buffer's size must be more than 4

◆ xl320_sendCommand()

xl320_sendCommand ( XL320_t *  xl320,
uint8_t  inst,
uint16_t  nbParams,
uint8_t *  params 
)

Send a command to the corresponding XL320. If this function is used according to the DYNAMIXEL PROTOCOLE 2.0 the address field is coded on 1 byte. The the parameters table should looks like the following example. The 0 stands for the most significant bits of the address which are not used.

uint8_t params[3] = {ADDR, 0, ..., ...}
Parameters
xl320XL320 device
instInstruction
nbParamsThe number of parameters. Can be 0
paramsThe pointer on the table containing the parameters. Can be NULL
